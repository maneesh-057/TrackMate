# 📊 Track Mate

Track Mate is a GUI-based Java application that helps users manage and track their monthly expenses. It allows setting a spending limit and alerts the user when the budget is exceeded. The application uses MySQL for data storage and JDBC for connectivity.

---

## ✅ Features

- Add and view daily/monthly expenses
- Set a monthly budget limit
- Alerts when expenses exceed the budget
- Clean, responsive GUI built with Java Swing/JavaFX
- Persistent data storage using MySQL
- Modular architecture with Model and DAO classes

---

## 🛠️ Requirements

- Java Development Kit (JDK 17 or later)
- IntelliJ IDEA / Eclipse / NetBeans (any Java IDE)
- MySQL Server (with MySQL Workbench preferred)
- Internet connection to install JDBC driver (if needed)
